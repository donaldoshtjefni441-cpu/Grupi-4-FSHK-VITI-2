# Personalized PageRank për një Rrjet Materialesh Kursi dhe Preferencash Tematike

## Përshkrimi

Ky projekt implementon algoritmin Personalized PageRank për një rrjet materialesh kursi dhe preferencash tematike.

Qëllimi është të demonstrohet se si preferencat e përdoruesit mund të ndikojnë në renditjen e nyjeve në një rrjet të orientuar.

## Struktura e Projektit

* `src/ranking.py` – implementimi i algoritmit PageRank.
* `Scripts/Personalized_PageRank.py` – ekzekutimi i eksperimentit dhe gjenerimi i grafikut.
* `figures/personalized_pagerank.png` – figura e gjeneruar nga programi.

## Instalimi

```bash
pip install -r requirements.txt
```

## Ekzekutimi

```bash
python Scripts/Personalized_PageRank.py
```

Pas ekzekutimit krijohet figura:

```text
figures/personalized_pagerank.png
```

## Libraritë e përdorura

* NumPy
* Matplotlib
